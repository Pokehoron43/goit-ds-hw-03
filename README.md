# Data Processing Projects (MongoDB Bot + Scraper)

## Project Overview
This repository contains two separate mini-projects:

1. Web scraping project for collecting and processing data  
2. MongoDB bot for interacting with the database  

## Project Structure

### 1. MongoDB Bot (`/Task_1`)
- Connects to MongoDB
- Performs CRUD operations
- Allows interaction via command input

### 2. Web Scraper (`/Task_2`)
- Scrapes data from a website
- Parses multiple pages
- Saves data into JSON files




## Tech Stack
- Python
- Requests / BeautifulSoup
- MongoDB
- JSON
- Urlib
- Pipenv


## Features
- Data scraping and parsing
- JSON data storage
- MongoDB integration
- CRUD operations via bot interface
